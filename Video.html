Actimux MULTIC-FS : Video description
The video in can be useed to config the FB size/Cature size/display position/display or not


1 - Parameters Description
Parameter Description
FrameBufferWidth The capture FB Width
if set all FB width size to 10, the video task will not be creat.
FrameBufferheight The capture FB Height
X_pos the horizontal start pos of capture video
X_resol the horizontal resolution of capture video
Y_pos the vertical start pos of capture video
Y_resol the vertical resolution of capture video
2 - Mnemonics Description
Mnemonic STraToN Type Description
Dis BOOL This var determine whether to display the video
State BOOL The video's diaplay state
Video_X WORD The video's X position
Video_Y WORD The video's Y position
Video_W BOOL The video's Width
Video_H BOOL The video's Height
ATTENTION :
  Video channel 1/3 use the same Video_X Video_Y Video_W Video_H
  Video channel 2/4 use the same Video_X Video_Y Video_W Video_H
